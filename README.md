# Stuff
This is a repo to work on minor projects.
As of now, it contains contains the [mxd_Renamer](mxd_Renamer.bat), which is a Windows Batch program to convert .mxd.doc files (Google Drive adds on the .doc extension) back to .mxd files. Simply download the program, place it in the folder containing your .mxd files and execute it (double click it)
